# Modern UI Demo (Bootstrap + React)

This is a small, modern, responsive React demo using Bootstrap components. It demonstrates component-based UI, folder structure, responsive layout, and a polished landing page style.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Commit and push to GitHub:

```bash
git init
git add .
git commit -m "Initial modern UI scaffold"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
